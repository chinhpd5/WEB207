Front end Framework

  1. Bootstrap là 1 framework giúp bạn thiết kết giao diện mạnh mẽ
    • Đơn giản mạnh mẽ
    • Thích nghi với mọi loại thiết bị
    • Được sử dụng phổ biến nhất
  2. Angular JS là 1 framework giúp bạn lập trình giao diện hiện đại nhất
    • Mã nguồn mở
    • Công nghệ mới
    • Hỗ trợ dịch vụ tương tác
